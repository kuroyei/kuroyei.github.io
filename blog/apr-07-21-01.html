共通する部分をJavaScriptで読み込ませる！

Apr 07, 2021

head、header、footerをJavaScriptから読み込むようにしたい

HTML

<header>
<a href="https://kuroyei.github.io/about-me.html" class="headline">Kuroyei</a>
<ul class="nav-list">
<li class="nav-list-item">
<a href="https://kuroyei.github.io/about-me.html">About me</a>
</li>
<li class="nav-list-item">
<a href="https://kuroyei.github.io/blog.html">Blog</a>
</li>
</ul>
</header>


上のHTMLコードを・・・

header.js

document.write('<header>・・・</header>')


上のように('')の中にぶち込む。すると・・・

HTML

<script src="https://kuroyei.github.io/common/header.js"></script>


毎回<header>・・・</header>をコピペせずとも上の一行のコードで済む
又、headerに変更を加えたい時はheader.jsを編集するだけで済む
